# Project Summary
The LegaBot project is a cutting-edge web application that serves as a landing page for "The Autonomous Football Universe," a fully AI-operated football ecosystem. It engages investors, fans, and potential partners by showcasing the transformative potential of AI in sports through a visually appealing interface that highlights its unique business model, revenue opportunities, and future roadmap. The project aims to revolutionize the way football is experienced and monetized.

# Project Module Description
The project consists of several functional modules that build the landing page:
- **Hero Section**: Introduces LegaBot with key visuals, animations, and calls to action. Recently updated for improved text visibility.
- **Thesis Section**: Outlines the core thesis of AI in sports with engaging content.
- **League Section**: Describes the autonomous match engine and player dynamics, featuring statistics and visuals.
- **Core Engine Section**: Details the AI systems powering the league, including interactive elements.
- **Digital Players Section**: Explains the characteristics and evolution of AI players.
- **Content Engine Section**: Highlights the media generated from matches, showcasing various content types.
- **Global Scale Section**: Presents worldwide engagement metrics and operational statistics.
- **Monetization Section**: Discusses revenue streams and business model potential.
- **Roadmap Section**: Outlines phases of development and key milestones in a visually structured timeline.
- **Demo Video Section**: Newly added section featuring an embedded YouTube demo video showcasing the project.
- **Footer Section**: Provides updated contact information with email links for investment inquiries and includes links to Terms, Policies, and Cookie Policy.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Vite**: Build tool for modern web applications.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **shadcn/ui**: UI component library for React.
- **Poppins**: Google Font used for typography.

# Usage
To get started with the project, follow these steps:
1. Clone the repository.
2. Navigate to the `app/frontend` directory.
3. Install dependencies:
   ```bash
   pnpm install
   ```
4. Build the project:
   ```bash
   pnpm run build
   ```
5. Run the development server:
   ```bash
   pnpm run dev
   ```
